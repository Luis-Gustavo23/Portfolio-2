## 👨‍💻 Sobre mim

Sou um desenvolvedor em formação com experiência em **HTML, CSS, JavaScript e C#**, focado na construção de interfaces modernas e funcionais.

Tenho vivência em áreas como **administrativo, logística e atendimento**, o que fortaleceu minhas habilidades de organização, comunicação e trabalho em equipe — características que aplico diretamente no desenvolvimento de software.

## 🎯 Sobre o projeto

Este portfólio foi desenvolvido com o objetivo de simular um cenário real de aplicação, onde os dados são consumidos de uma fonte externa (**JSON**), aproximando o projeto de uma arquitetura utilizada no mercado.

A aplicação apresenta minhas informações de forma dinâmica, organizada e interativa.

## 💡 Diferenciais técnicos

✔️ Consumo de dados externo com `fetch()`
